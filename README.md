# 📝 Django TODO Application

A simple and user-friendly TODO (Task Manager) web application built using Django.  
This project allows users to manage their daily tasks efficiently with full CRUD functionality.

---

## 🚀 Features

- Add new tasks  
- View all tasks  
- Update/edit tasks  
- Delete tasks  
- Mark tasks as completed  
- Clean and responsive UI  

---

## 🛠️ Tech Stack

- **Backend:** Python, Django  
- **Frontend:** HTML, CSS, Bootstrap  
- **Database:** SQLite
